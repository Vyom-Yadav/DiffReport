 * Checks that switch statement has a {@code default} clause.
30 
 * </p>
31 
 * <p>
32 
 * Rationale: It's usually a good idea to introduce a
33 
 * default case in every switch statement. Even if
34 
 * the developer is sure that all currently possible
35

36

37

38

39

40
